# Novaro Systems Website

A modern Django-based company website for **Novaro Systems**, built to showcase services, portfolio work, client testimonials and allow users to contact the team through a working contact form.

## Project Overview

Novaro Systems is a digital solutions website designed with a clean, modern and responsive layout. The project includes multiple pages, reusable template structure, static files management, contact form submission and Django admin integration for managing submitted messages.

This project was created as a professional company website and also as a portfolio project to demonstrate Django web development, frontend design and basic backend form handling.

## Features

* Responsive company website
* Clean modern UI design
* Reusable `base.html` template structure
* Home page with hero section
* About page with company and team information
* Services page
* Portfolio page
* Testimonials page
* Contact form with database submission
* WhatsApp direct contact button
* Django admin panel customization
* Static files organized properly
* Mobile-friendly navigation menu
* Smooth animations and interactions using JavaScript
